  • 组件内发请求只需引入 axios 二次封装的文件即可，如下面的代码需要在发送请求的组件内引入 requests，不需再引入 axios

  • axios 是发送请求的服务，采用 axios 的方式的话，哪里用到发送请求都需要用到 axios，二次封装的目的是为了添加请求和响应拦截器

  • 前端向服务器发送请求的方法：XMLHttpRequest、fetch、JQ、axios；

  • 为什么要进行二次封装 axios？

​ 答：添加请求拦截器和响应拦截器，请求拦截器可以在发请求之前处理一些业务，响应拦截器当服务器数据返回以后可以处理一些事情。

  • 项目的 api 文件夹下存放处理 axios 请求的文件

封装代码：requests.js 请求真实数据

  • mock 的 axios 二次封装：mockRequests.js 请求 mock 虚拟数据
  • mock 的 API 接口管理：index.js
